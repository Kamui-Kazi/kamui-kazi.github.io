Stream Elements

Add/Update Commands

ADD: !command add commandName Enter the message you want.
Example: !command add medic Medic is the best.
EDIT: !command edit commandName Enter the updated message.
Example: !command edit medic Medic has left the chat.

Popular Commands

  • !changeservers
  • !collab - list the players Siri is playing with.
  • !delay - explains how to check their stream delay
  • !qsnipe - list the servers/game mode Siri is playing.
  • !raid - Raid message for subs
  • !raid1 - Raid message for subs
  • !fraid - Raid message for followers
  • !pine - Soft shoutout for Pine
  • !raiders - Welcome new raiders
  • !setgame - Updates the Twitch stream category
  • !topic - change the topic of chat

Timer Commands

  • Horror Stream Warning: !timer enable/disable hsw
  • SiriBot Timer: !timer enable/disable siribot

Twitch Native

  • /announce - Have your message as an announcement in chat
  • /ban - ban a user (alternate way from pressing the ban buttons)
  • /unban - unbans a user
  • /pin - Pin your message
  • /prediction - opens the prediction creation UI
  • /poll - opens the poll creation UI
  • /requests - opens the rewards request queue
  • /shield - enables shield mode
  • /shoutout - shouts out a user (WE DO NOT USE !SO)
  • /timeout - times out a user
  • /user - displays user information
  • /warn - warns a user
