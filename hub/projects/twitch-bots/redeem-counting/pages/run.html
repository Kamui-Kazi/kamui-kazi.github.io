Running the bots

The next step is to run your bots, this is all you should need to do every future time you run these bots.

There are 2 steps to running your bots, don't worry it's all done through a menu:

  1. run the file "run.py"
  2. on the menu screen, select option 1

when the bots are running you should see logging events in the console when:

  • a command is used
    • logging who and what was used
    if related to the count will indicate the amount input and the new count
  • an unknown command is used
    • logging who and what was used
  • a redeem is made (optional, disabled by default)

The bots should now be working, but I recommend you look at the usage guide to see what the bots can do. Next
