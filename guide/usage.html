Command name !meow x alias arguments description
Count !meows !meow count / !meow none Replies to the user with the current count of meows. Is also the default for the !meow x command
Rewards !meow_rewards !meow rewards none Replies to the user with the cost of each payout
payouts table
Command List !meow_commands !meow commands none Replies to the user with the commands they have the permission to use
Add Meows !add_meows !meow add integer (default 0) Replies to the user with the new count of meows.
The integer can be any whole number from 0 - 9223372036854775807,
if no integer is provided the program defaults to 0
Subtract Meows !sub_meows !meow sub integer (default 0) or string (default 0) Replies to the user with the new count of meows.
Set Meows !set_meows !meow set integer (default 0) Replies to the user with the new count of meows.
Reset Meows !reset_meows !meow reset none Resets the meow count to 0
