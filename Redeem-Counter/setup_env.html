Redeem Counter Bot - Setup Guide - Environment Variables

The next step is to provide the program with the information it needs to run as your bot.
We do this by using what are called "environment variables" these are stored in a file named ".env"
To set these there are 2 main methods provided in this program,

  1. Set it up using the menu provided with the bot
  2. Set it up using the example provided in ".env.example"
both of these methods will require some information
  • your twitch app's client ID
  • your twitch app's client secret
If you don't have the information nessisary then go back to the previous page and setup a twitch app.
If you do have that information then you can proceed.

Getting you twitch ID numbers

In this section we will be getting the ID numbers for our twitch accounts,
this is what the bot uses to connect to your accounts.
This is nessisary no matter if you are using the menu or doing it manualy

  1. visit the site "https://www.streamweasels.com/tools/convert-twitch-username-%20to-user-id/"
  2. in the twitch username field enter your twitch bot's username
  3. click "Convert Twitch Username to ID"
  4. save the number generated, this is the ID for your bot's account
  5. in the twitch username field enter your streamer's twitch username
  6. click "Convert Twitch Username to ID"
  7. save the number generated, this is the ID for your streamer's account
With that done you should now have all the information to fill in the ".env"

Menu guided setup

To begin I will guide you through making the ".env" using the menu in the bot

  1. Run the file called run.py this should bring up a menu screen
  2. Select option 3 by typing 3 and pressing enter
  3. The first thing it asks for is your client ID, paste that in and press enter
  4. Then it will ask for your client Secret, paste that in and press enter
  5. Then it will ask for the streamer's name, paste that in and press enter
  6. Then it will ask for the Streamer's ID, paste that in and press enter
  7. Then it will ask for the Bot's name, paste that in and press enter
  8. Then it will ask for the Bot's ID, paste that in and press enter
This is all the information that the bot will need at this moment. you are now ready for the next step
