Usage Table

Command name !meow x alias arguments (default) description
Count !meows !meow count / !meow none Replies to the user with the current count of meows. Is also the default for the !meow x command
Rewards !meow_rewards !meow rewards none Replies to the user with the cost of each payout
payouts
Command List !meow_commands !meow commands none Replies to the user with the commands they have the permission to use
Add Meows !add_meows !meow add number (0) Replies to the user with the new count of meows.
you can provide a positive whole number
Subtract Meows !sub_meows !meow sub number or name (0) Replies to the user with the new count of meows.
you can provide a positive whole number,
or you can provide a name that corosponds to a payout,
ie. "!meow sub nya" is the same as "!meow sub 300"
a list of these names can be found on the payouts table
Set Meows !set_meows !meow set number (0) Replies to the user with the new count of meows.
Reset Meows !reset_meows !meow reset none Resets the meow count to 0
